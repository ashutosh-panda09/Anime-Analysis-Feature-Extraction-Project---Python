# Anime Dataset Analysis
I did a project where I looked at a bunch of anime data. I used Python and some other tools like Pandas and Jupyter Notebook to clean up the data and see what I could learn from it.

## Project Overview
This project was about taking some data about anime and turning it into something that I could actually use to answer some questions. I had 50 anime records and 3 features to work with. I wanted to see if I could figure out some things about anime ratings, episode counts and when they were released.

## Objectives
Here are the things I wanted to do:
- Look at the data and understand what it was about.
- Get the episode counts out of the messy text.
- Figure out how long each anime lasted in months.
- Find the anime based on some criteria.
- Practice using Pandas and writing my own Python functions.

## Dataset
My dataset had 50 rows and 3 columns.
The columns I used were:
- Rank
- Title
- Score
    
## Tools & Technologies
I used the following tools:
- Python
- Pandas
- Jupyter Notebook

## Key Techniques Used
I didn't a lot of things to the data. But Here Are Some:

- Checking the data types and converting them when necessary
- Extracting and transforming text
- Parsing dates and calculating durations
- Filtering the data based on conditions
- Sorting the data with `sort_values()`
- Ranking the data and picking the top records
- Doing some basic exploratory data analysis
